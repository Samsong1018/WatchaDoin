# Whatcha Doing? Style Guide

## Inline Styles Policy
Inline styles are avoided. All global styles live in /css/styles.css.
Page-specific styles (if any) are placed in the <head> of that page only.
Inline styles are not used in this project.

## Color Palette
- Background: #f5f7fa (light gray)
- Surface: #ffffff (white)
- Accent: #4a7cf7 (blue)
- Text: #1a1f2e (near black)
- Muted text: #5a6480

## Typography
- Font: Segoe UI / system-ui
- Base size: 1rem (16px)
- Hero: clamp(2rem, 5vw, 3.5rem)

## Design Direction
Clean, focused productivity aesthetic — light and spacious with a reliable blue accent.
